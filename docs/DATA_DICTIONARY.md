# Data Dictionary

- `sol_close`: dagelijkse UTC-slotprijs SOL.
- `btc_close`: dagelijkse UTC-slotprijs BTC.
- `relative_strength_btc_7d`: SOL-rendement over 7 dagen min BTC-rendement.
- `dex_volume_ratio_7d_30d`: 7-daags DEX-volume ten opzichte van 30-daags normaal.
- `evidence_quality`: transparante score voor de stevigheid van de conclusie.
