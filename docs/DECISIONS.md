# Decisions

- Geen LLM voor dagelijkse conclusies: teksten zijn deterministisch.
- Marktsignaal en bewijskwaliteit blijven gescheiden.
- Marktbreedte blijft experimenteel totdat voldoende dagelijkse snapshots bestaan.
- Slechte backtestresultaten worden zichtbaar getoond.
- In deze Codex-omgeving was lokaal klonen van de private repo zonder interactieve GitHub-auth niet mogelijk; de implementatie is lokaal gebouwd en is geschikt voor publicatie naar de lege repo.
