# Limitations

Historische relaties kunnen veranderen. Crypto kent regimewisselingen. API-data kan worden herzien. TVL en volume kunnen door prijsbewegingen worden beïnvloed. Tokenbreedte kan survivorship bias bevatten. Historische analogieën bewijzen geen causaliteit. Kleine steekproeven geven brede onzekerheid. Backtests zijn geen garantie.
