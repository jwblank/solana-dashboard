# Data Sources

Coinbase Exchange levert dagelijkse SOL-USD en BTC-USD candles. CoinGecko levert actuele prijscontrole en ecosysteembreedte. DeFiLlama levert TVL, stablecoins, DEX-volume en fees. Solana RPC levert compacte actuele netwerkcontext die niet als historisch gevalideerd signaal wordt gebruikt.

Productiemodus gebruikt geen fixtures. Bij bronuitval mag alleen laatst bekende geldige data worden gebruikt en moet de bewijskwaliteit dalen.
