# GitHub Pages Setup

Voor een private repository hangt GitHub Pages af van het GitHub-abonnement.

Opties:

1. Houd de repository privé met een abonnement dat Pages voor private repositories ondersteunt.
2. Maak de repository openbaar wanneer publieke GitHub Pages gewenst is.
